# My To-Do List Application

A simple and beautiful To-Do List application built with React and Vite.

## Github-Repository-Link: https://github.com/Prateek-saraswat/Todo_Application


## Features

- Add new tasks
- Mark tasks as completed
-  Edit existing tasks
-  Delete tasks
- Beautiful dark-themed UI with Tailwind CSS

## Tech Stack

- React
- Vite
- Tailwind CSS

## How to Run

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and go to:
```
http://localhost:5173
```

## Project Structure

```
src/
├── App.jsx              # Main component with state management
├── components/
│   ├── Header.jsx       # App header
│   ├── AddTodo.jsx      # Form to add new todos
│   ├── TodoList.jsx    # Renders list of todos
│   └── TodoItem.jsx    # Individual todo with edit/delete
```
